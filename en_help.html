Conway's Game Of Life

The Game Of Life is a cellular automaton, devised by John Horton Conway in 1970.

Rules Of Life:

1. Any live cell with fewer than two live neighbours dies, as if caused by underpopulation.
2. Any live cell with two or three live neighbours lives on to the next generation.
3. Any live cell with more than three live neighbours dies, as if by overpopulation.
4. Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.

Navigation:

Left mouse button (clicked): Inverts state of the clicked cell.
Left mouse button (hold): Revives all hovered cells.
Right mouse button (clicked): Kills the clicked cell. If no cell is clicked, the context menu will be shown.
Right mouse button (hold): Kills all hovered cells.
Mouse wheel (clicked): Shows context menu if a cell is clicked.
Mouse wheel (scrolled): Moves the view if not the whole area can be shown.
[CTRL] + mouse wheel (scrolled): Zoom (+/-)
Arrow keys: Moves the view if not the whole area can be shown.
[CTRL] + [+]: Zoom in
[CTRL] + [-]: Zoom out
