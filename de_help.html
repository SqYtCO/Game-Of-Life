Conways Spiel des Lebens

Das Spiel des Lebens ist ein cellulärer Automat, der 1970 von dem Mathematiker John Horton Conway entworfen wurde.

Regeln des Lebens:

1. Eine lebende Zelle mit weniger als zwei lebenden Nachbarn wird in der nächsten Generation an Einsamkeit sterben.
2. Eine lebende Zelle mit zwei oder drei lebenden Nachbarn wird in der nächsten Generation weiterleben.
3. Eine lebende Zelle mit mehr als drei lebenden Nachbarn wird in der nächsten Generation an Überbevölkerung sterben.
4. Eine tote Zelle mit genau drei lebenden Nachbarn wird in der nächsten Generation neu geboren.

Navigation:

Linke Maustaste (geklickt): Kehrt den Status der Zelle um.
Linke Maustaste (gehalten): Belebt alle Zellen, über die gefahren wird.
Rechte Maustaste (geklickt): Tötet die angeklickte Zelle. Wenn keine Zelle angeklickt wird, wird das Kontextmenü angezeigt.
Rechte Maustaste (gehalten): Tötet alle Zellen, über die gefahren wird.
Mausrad (geklickt): Zeigt das Kontextmenü, wenn auf eine Zelle geklickt wird.
Mausrad (gescrollt): Bewegt die Ansicht auf dem Feld, wenn es nicht ganz angezeigt werden kann.
[STRG] + Mausrad (gescrollt): Zoom
[STRG] + [+]: Vergrößern
[STRG] + [-]: Verkleinern
